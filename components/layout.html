# layout-布局

协助进行页面级整体布局。

# 组件注册

<script>
    import {Layout,Header,Footer,Content,Sider} from "origin-ui"
    export default {
        components:{
            "o-layout":Layout,
            "o-header":Header,
            "o-footer":Footer,
            "o-content":Content,
            "o-sider":Sider,
        }
    }
</script>

# 基本使用

效果：

默认布局
带有侧边栏布局
侧边栏靠右
左右布局
